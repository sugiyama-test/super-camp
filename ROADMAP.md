## Phase 3: AWSデプロイ

### 3-1. 本番用ビルド
- [ ] backend/Dockerfile.prod（マルチステージビルド）
- [ ] frontend/Dockerfile.prod（standalone出力）
- [ ] next.config.ts に `output: "standalone"` 追加
- [ ] CORS 環境変数化（`CORS_ALLOWED_ORIGINS`）

### 3-2. AWSインフラ
```
[ユーザー] → [ALB] ─ /api/* → [ECS Fargate: backend]
                   └ /*     → [ECS Fargate: frontend]
                                      ↓
                           [RDS PostgreSQL]
```
- [ ] ECR リポジトリ（backend / frontend）
- [ ] VPC + サブネット構成
- [ ] RDS PostgreSQL（db.t3.micro）
- [ ] ECS Fargate クラスター + サービス
- [ ] ALB（パスベースルーティング）
- [ ] コスト目安: ~$52/月（無料枠適用で削減可能）

### 3-3. CI/CD
- [ ] GitHub Actions: テスト → ECR push → ECS デプロイ
- [ ] デプロイ時にマイグレーション自動実行
